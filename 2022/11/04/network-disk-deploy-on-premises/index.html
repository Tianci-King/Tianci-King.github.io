网盘挂载到本地的部署方法

网盘 Alist RaiDrive
爱瞎折腾
发布日期:   2022-11-04
更新日期:   2023-10-06
文章字数:   794
阅读时长:   3 分
阅读次数:  

导言

由于我使用云盘的次数很频繁，每次想存入或取出文件都需要打开软件、登录、操作、退出……十分繁琐

于是我采用网盘挂载到本地的部署方法将云盘直接映射到此电脑中，可以快捷地使用我的云盘，彻底舍弃掉了云盘软件

并且，这样的作法变相的相当于扩容了我的硬盘，我可以将一些轻量的文件或是数据存入其中

当然，从该映射的”磁盘“中存入或是读取，取决于云盘的带宽和你的网速

本方法基于Alist与RaiDrive，软件自行下载，地址如下：

​ Alist：

​ Releases · alist-org/alist (github.com)

​ （按照自己电脑的版本下载对应的Alist.exe文件，注意如果是3.0及以上版本不能 直接双击安装，而需要在cmd中操作，具体步骤见下）

​ RaiDrive：

​ https://forspeed.rbread05.cn/down/newdown/5/28/RaiDrive.rar

​ （双击安装，无脑~）

步骤一：Alist

1.打开cmd

2.切换路径至刚刚下载的Alist.exe所在的目录下

3.输入

alist.exe

完成安装

4.再输入

alist.exe server

将会启动Alist，请复制倒数第三行末尾的password，接下来要用

运行之后切记，cmd窗口不能关闭，暂时先开着

5.访问Alist主页

http://127.0.0.1:5244/@manage

输入用户名和密码：用户名默认为admin，密码为刚刚cmd中复制的password

（进入主页后可以修改用户名和密码

步骤二：获取token进行网盘添加

1.打开

Guide | AList文档 (nn.ci)

2.在左侧菜单栏添加存储里选择自己要添加的网盘（这里以百度网盘为例）

按以下步骤进行操作

3.获取刷新令牌后回到刚刚的http://127.0.0.1:5244/@manage主页

添加成功以后在主页就会出现该标签，状态：work

步骤三：RaiDrive

1.安装完成后，进入setting切换设置

2.进入添加添加网盘

其中虚拟驱动盘和盘名可自行设定

然后在本地就能看到该盘了

后记

不足之处

然而为了确保网盘本地部署的正常运行，需要在后台挂Alist和RaiDrive两个程序。

RaiDrive尚可缩小到托盘，不影响使用体验，而Alist则必须挂在cmd中，很影响观感

所以可以采用自启动Alist的方法，无需每次都在cmd中输入Alist.exe server的繁琐操作，仅在需要挂载本地网盘时打开RaiDrive软件即可

改进方法

1.在alist.exe所在的文件夹的同一个目录下

创建2个txt文件，分别取名为启动.txt和停止.txt

2.在启动.txt中输入

Set ws = CreateObject("Wscript.Shell") 
ws.run "alist.exe server",vbhide

在停止.txt中输入

Dim Wsh
Set Wsh = WScript.CreateObject("WScript.Shell")
Wsh.Run "taskkill /f /im alist.exe",0
Set Wsh=NoThing
WScript.quit

3.修改2个文件的后缀为.vbs

4.win+R键：输入shell:Common Startup

将启动.vbs创建快捷方式放入该目录下

以上就是将网盘挂载到本地部署的全部内容~
